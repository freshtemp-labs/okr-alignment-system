# 贡献指南

感谢你对本项目的兴趣！这是一个由 AI Agent（Hermes Agent + 土鳖）协作开发的项目，我们欢迎任何形式的贡献。

## 🚀 快速开始

1. Fork 本仓库
2. Clone 到本地：`git clone https://github.com/你的用户名/okr-alignment-system.git`
3. 用 Xcode 打开 `project/OKRAlignment.xcodeproj`
4. 选择 iOS 或 macOS target，Build & Run

## 🐛 报 Bug

- 使用 [GitHub Issues](https://github.com/freshtemp-labs/okr-alignment-system/issues) 提交
- 请包含：复现步骤、期望行为、实际行为、截图（如有）

## 💡 提功能

- 先开 Issue 讨论你的想法
- 获得认可后再开始实现

## 🔧 开发流程

1. 从 `main` 创建分支：`git checkout -b feat/你的功能名`
2. 修改代码，确保编译通过
3. 运行测试：`Cmd+U` in Xcode
4. 提交 PR，描述你做了什么

## 🏷 Good First Issues

查看标有 [`good first issue`](https://github.com/freshtemp-labs/okr-alignment-system/labels/good%20first%20issue) 的 Issue，这些是适合新贡献者的任务。

## 📐 代码规范

- Swift 代码遵循 SwiftLint 规范
- 使用 SwiftUI 最佳实践
- 注释使用中文或英文均可
- 保持 MVVM 架构分层

## 🤖 关于 AI 协作开发

本项目的核心代码由 Hermes Agent 和土鳖（DeepSeek V4）协作编写。你可能会看到一些 AI 生成的代码模式——这正是我们想要改进的地方。如果你发现代码可以更"人性化"，请大胆修改！

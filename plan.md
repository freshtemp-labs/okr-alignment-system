# OKR Alignment System - 软件开发计划

## 项目概述
基于用户提供的HTML Demo，开发一套完整的"企业与个人OKR级联对齐管理系统"。系统支持macOS和iOS双平台，核心特性包括树状可视化、级联进度计算、多用户协作等。

## 技术方案
- **框架**: SwiftUI (跨macOS/iOS共享代码)
- **架构**: MVVM + Clean Architecture
- **数据持久化**: Core Data + CloudKit (同步)
- **最小支持版本**: macOS 14, iOS 17
